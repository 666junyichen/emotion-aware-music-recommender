# Static Assets

This public repository does not include private or license-sensitive original media assets.

It includes three generated WAV files for the public demo:

- `static/audio/demo-happy.wav`
- `static/audio/demo-sad.wav`
- `static/audio/demo-calm.wav`

These files are synthetic placeholder audio clips created for the public demo. They are not the original tracks listed in `music.json`.

To enable full playback, add allowed assets under these paths:

- `static/audio/`
- `static/album/`
- `static/images/`
- `static/icons/`

The file paths should match the values in `music.json` and `app.py`.
