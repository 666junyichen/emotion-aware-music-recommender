# 项目说明

## 基本信息

项目名称：PulseScape / Zen Music Recommender

项目阶段：情绪感知音乐推荐原型，已升级为可部署的作品集 Web Demo

线上地址：https://emotion-aware-music-recommender.vercel.app

主要技术：Python, Flask, Jinja, JSON, HTML, CSS, JavaScript, Vercel

建模相关技术：NeuroKit2, pandas, scikit-learn, Random Forest

## 项目是什么

PulseScape 是一个根据用户当前心率 BPM 推荐音乐的 Web 应用。用户输入心跳频率后，系统会推断当前状态，再从音乐库中选择一首适合的歌曲，并展示推荐理由、沉浸式情景背景、相似歌曲和收藏功能。

项目原型来自 Zen Music Recommender：一个结合 ECG/HRV 特征、情绪分类实验和音乐推荐流程的课程/作品集项目。现在已经升级成更完整的 Web Demo。

## 当前功能

- 输入当前 BPM。
- 根据 BPM 映射到 Calm、Sad、Happy、Focused、Recovery 等状态。
- 推荐一首音乐。
- 播放器显示歌曲标题、艺术家、BPM、energy、valence、genre 和 vocal type。
- 展示推荐理由。
- 展示类似音乐。
- 用户可以点击类似音乐或音乐库卡片进入播放器。
- 用户可以收藏歌曲，收藏结果保存在浏览器 `localStorage`。
- 用户可以点击 Like / Not for me，当前会话内会影响后续推荐排序。
- 用户可以输入听后 BPM，用于记录音乐后的状态变化。
- 音乐库页面展示所有歌曲 metadata。
- 当前新增的 10 首 public MP3 可以直接播放；原始项目音频如果不可公开或不存在，会播放 demo fallback audio。

## 推荐逻辑

现在的推荐不再是随机选歌，而是基于透明评分：

```text
score =
  emotion_match * 40
+ bpm_closeness * 30
+ energy_match * 15
+ valence_match * 10
+ small_random_variation
```

这样既能保持推荐系统的可解释性，也能让 demo 比简单随机推荐更像真实产品。

## 背景图状态

当前每首歌都有 `scene_image` 字段，先使用稳定的远程占位图，保证每首歌视觉上不完全一样。

后续可以用 Pexels 或 Pixabay API 自动替换为更准确的情景图片。项目已经新增脚本：

```text
scripts/fetch_scene_images.py
```

Pexels 用法：

```powershell
$env:PEXELS_API_KEY="your-key"
python scripts/fetch_scene_images.py pexels
```

Pixabay 用法：

```powershell
$env:PIXABAY_API_KEY="your-key"
python scripts/fetch_scene_images.py pixabay
```

脚本会把图片 URL、来源页、作者和 license 写回 `music.json`。

推荐优先用 Pexels API。Pexels 免费 API key 默认额度足够这个项目使用，注册后把 key 放到环境变量即可。Pixabay 也可以用，但部分页面下载会遇到 Cloudflare 验证。
